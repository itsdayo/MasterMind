The Rules of MasterMind

MasterMind is a two player game or a game played against the computer. We have one player as the codemaker trying to create a difficult combination of 4 colors, duplicates allowed, so that the second player, the codebreaker will have trouble guessing the pattern. After each guess the codebreaker makes they're color coded pegs that grades each guess. Red represents a color in the correct spot White represents a color used correctly but not in the correct spot and black represent a color not used.

  • The modern game with pegs was invented by a man named Mordecai Meirowitz, a telecommunications expert. It resembles a similar an earlier game called Bulls and Cows.

  • Many mathematicias play and try to create what is called a Five Guess algorithm, which is an algorithm that will crack any code in a number a alittle less than 5.

  • David Knuth showed that his codebreaking algorithm can solve anyones code in 5 turns or fewer by decreasing the numbr of possible patterns.
  • Create a set of the number of possibilites, there are 1296 in the beginning. The first guess most be an aabb which means to colors being duplicated and next to each other.
  • Remove all the combinations that will not give you a better peg score in the set of the number of possibilities.
  • After each guess calculate the amount of possiblities you can have less and use the guess that will give you the the greater peg score.
  • Go back to step two until you break the code
